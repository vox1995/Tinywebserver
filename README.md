TinyWebServer
===============
Linux下C++轻量级Web服务器，可以用于网络编程的实践，搭建属于自己的服务器。

* 使用**线程池 + epoll(ET和LT均实现) + 模拟Proactor模式**并发模型
* 使用**状态机**解析HTTP请求报文，支持解析**GET和POST**请求
* 通过访问服务器数据库实现web端用户**注册、登录**功能，可以请求服务器**图片和视频文件**
* 新增了web用户的**注销**功能，可以实现已注册用户的账号注销
